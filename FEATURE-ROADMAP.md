## 2026年6月开发计划

### P0 — 核心卖点（小红书引流直接相关）

#### 1. 接入真实 A 股数据

- **参考**: daily_stock / go-stock
- **数据源**: 东方财富 push2 API、AkShare、Tushare
- **内容**: 实时行情、财报指标（RoE/毛利率/净利率等）、K线数据
- **目的**: 替换 Mock 数据，建立可信度
- **影响**: 筛选器、金字塔、对比、AI 分析 全部受益

#### 2. 个股深度 AI 分析报告

- **参考**: hot-money (UZI-Skill)
- **能力**:
  - AI 自动采集 22 维数据
  - 多维度评分（财务/技术/情绪）
  - 生成结构化 HTML 报告（含评分卡、DCF 估值、风险提示）
  - **生成朋友圈竖图（1080×1920）** — 用户可直接发小红书
- **目的**: 核心卖点，用户付费 + 自发传播

#### 3. AI 多 Agent 评审团

- **参考**: hot-money 的 51 位投资大佬 + daily_stock 的 15 种策略
- **能力**:
  - 多个 AI Agent 从不同角度分析（价值投资、技术分析、游资视角等）
  - 综合打分 + 多空辩论
  - 给出最终建议
- **目的**: 差异化卖点，"多位投资大师帮你分析"

### P1 — 用户体验

#### 4. K 线图表 & 技术指标

- **参考**: go-stock / AI-Kline
- **能力**: 日K/周K/月K、MACD、KDJ、RSI、布林带
- **工具**: ECharts / TradingView Lightweight Charts
- **目的**: 满足技术分析用户需求

#### 5. 资金流向 & 龙虎榜

- **参考**: go-stock
- **能力**: 主力资金净流入/流出、北向资金、龙虎榜数据
- **目的**: 增加数据维度，吸引短线交易者

#### 6. 每日市场早报

- **参考**: daily_stock
- **能力**:
  - GitHub Actions 每日定时运行
  - AI 生成大盘复盘 + 板块涨跌 + 今日关注
  - 推送到网站 + 可订阅推送（微信/飞书/邮件）
- **目的**: 提升日活，增加用户粘性

### P2 — 商业化

#### 7. 会员注册 & 登录系统

- 邮箱注册 / 微信扫码登录
- 免费用户：每日 3 次 AI 分析
- 会员：无限次 + 深度报告 + 高清分享图

#### 8. 支付集成

- 微信支付 / 支付宝
- 参考当前定价：¥368 永久会员

#### 9. 使用次数 & 权限管理

- API 频率限制按会员等级区分
- 会员专属功能标记

## 技术栈建议

| 领域           | 推荐                                                |
| -------------- | --------------------------------------------------- |
| **数据源**     | 东方财富 push2 API（免费）、AkShare（Python，免费） |
| **AI 模型**    | DeepSeek（性价比最高）/ 硅基流动（免费额度）        |
| **图表**       | TradingView Lightweight Charts / ECharts            |
| **自动化**     | GitHub Actions（零成本定时任务）                    |
| **推送**       | PushPlus（微信推送，免费）/ Server酱                |
| **分享图生成** | Puppeteer / node-canvas 生成 1080×1920 图片         |
